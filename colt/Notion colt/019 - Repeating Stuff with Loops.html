204. Iterating Over Objects

In JavaScript, you can iterate over objects using various techniques. Here are a few common methods:

  1. for...in loop: The for...in loop is commonly used to iterate over the properties of an object. It iterates over all enumerable properties of an object, including inherited ones.
const obj = { a: 1, b: 2, c: 3 };

for (let key in obj) {
  console.log(key, obj[key]);
}
  1. Object.keys() method: The Object.keys() method returns an array of all enumerable property names of an object. You can then iterate over this array using a for loop or any array iteration method.
const obj = { a: 1, b: 2, c: 3 };

Object.keys(obj).forEach(key => {
  console.log(key, obj[key]);
});
  1. Object.entries() method: The Object.entries() method returns an array of an object's enumerable property key-value pairs. Each key-value pair is represented as a two-element array, and you can iterate over it using a for...of loop or any array iteration method.
const obj = { a: 1, b: 2, c: 3 };

for (let [key, value] of Object.entries(obj)) {
  console.log(key, value);
}

These are just a few examples of how you can iterate over objects in JavaScript. Choose the method that best suits your needs based on the specific requirements of your code.
