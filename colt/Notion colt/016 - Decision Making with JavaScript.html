165. Console, Alert, & Prompt

In JavaScript, the "console" is an object that allows you to interact with the browser's developer console. It provides various methods for displaying information and debugging your code.

Similar to how the "Math" object has different methods like "ceil", "random", "floor", and "pow", the "console" object also has its own set of methods. Some commonly used methods include:

  1. console.clear(): Clears the console window.
  1. console.error(): Prints an error message to the console.
  1. console.warn(): Prints a warning message to the console.
  1. console.time(): Starts a timer to measure the duration of an operation.
However, the most frequently used method is console.log(). It allows you to log messages or values to the console, which is useful for debugging and understanding how your code is executing. By default, log messages are displayed as regular output in the console.
You can also use console.warn() and console.error() to print warning and error messages respectively. They have a different appearance in the console, making it easier to identify and differentiate them from regular log messages.
So, in your JavaScript code, console.log() will be the primary method you'll use to output information during development.
