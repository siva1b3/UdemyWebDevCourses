Full web stack

front-end

HTML:
HyperText Markup Language - the standard language for creating web pages and web applications.
CSS:
Cascading Style Sheets - used for styling the presentation of HTML elements in web pages.
Bootstrap:
A popular CSS framework that simplifies creating responsive and visually appealing web designs.
SCSS (Sass):
Syntactically Awesome Stylesheets - a preprocessor scripting language that is interpreted or compiled into CSS.
Tailwind CSS:
A utility-first CSS framework that allows for rapid development of custom designs using pre-built utility classes.
React:
A JavaScript library for building user interfaces, often used for creating interactive and dynamic frontend components.
Angular:
A comprehensive front-end framework by Google for building dynamic and robust web applications.
Vue.js:
A progressive JavaScript framework for building user interfaces, known for its simplicity and flexibility.

Back end

Node.js:
A JavaScript runtime environment that allows for server-side execution of JavaScript code.
Express.js:
A minimalist web application framework for Node.js, commonly used for building server-side applications and APIs.
Next.js:
A React framework for building server-rendered applications and websites.
Django:
A high-level Python web framework that facilitates rapid development and clean, pragmatic design of web applications.
Flask:
A lightweight and flexible Python web framework for building web applications.

remaining parts of the full stack

Database (MS SQL Server):
Microsoft's relational database management system, used for storing and managing structured data.
ORM (Object-Relational Mapping):
Libraries like Sequelize (for Node.js) or SQLAlchemy (for Python) to interact with databases using object-oriented programming.
RESTful APIs:
Designing and implementing APIs following REST principles for efficient and standardized communication between the frontend and backend.
Version Control:
Tools like Git for tracking changes in code, collaborating with other developers, and managing different versions of the codebase.
Package Managers:
Tools like npm (for Node.js) or pip (for Python) to manage and install project dependencies efficiently.
Testing Frameworks:
Using testing frameworks like Jest (for JavaScript), Pytest (for Python) to ensure code reliability and functionality.
Deployment & DevOps:
Configuring and automating deployment processes using tools like Docker, Kubernetes, and CI/CD pipelines to enhance efficiency and scalability.
Versioning & Environment:
Utilizing tools like Docker for containerization and virtualization to manage different environments and versions of the application.
