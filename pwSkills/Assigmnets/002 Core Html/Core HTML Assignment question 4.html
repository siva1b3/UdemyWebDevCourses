1. <a>: Defines a hyperlink.
  2. <abbr>: Defines an abbreviation or acronym.
  3. <area>: Defines an area inside an image map.
  4. <article>: Defines an article.
  5. <body>: Contains the main content of an HTML document.
  6. <br>: Inserts a single line break.
  7. <button>: Defines a clickable button.
  8. <col>: Specifies column properties for each column within a <colgroup> element.
  9. <colgroup>: Specifies a group of one or more columns in a table for formatting.
  10. <del>: Defines text that has been deleted from a document.
  11. <dfn>: Defines a definition term.
  12. <div>: Defines a division or section in an HTML document.
  13. <dl>: Defines a description list.
  14. <dt>: Defines a term/name in a description list.
  15. <em>: Defines emphasized text.
  16. <embed>: Embeds external content or plugin within an HTML document.
  17. <fieldset>: Groups related elements in a form.
  18. <figcaption>: Defines a caption for a <figure> element.
  19. <figure>: Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
  20. <footer>: Defines a footer for a document or section.
  21. <form>: Defines an HTML form for user input.
  22. <frame>: Defines a window (a frame) in a frameset.
  23. <h1> to <h6>: Defines HTML headings.
  24. <head>: Contains meta-information about the document.
  25. <header>: Defines a header for a document or section.
  26. <hr>: Defines a thematic break, such as a paragraph change or a thematic shift in content.
  27. <html>: Defines the root of an HTML document.
  28. <i>: Defines italic text.
  29. <iframe>: Defines an inline frame, used to embed external content into a web page.
  30. <img>: Embeds an image.
  31. <input>: Defines an input field where the user can enter data.
  32. <legend>: Defines a caption for a <fieldset> element.
  33. <li>: Defines a list item within a list.
  34. <link>: Defines relationships between a document and an external resource (most used to link to style sheets).
  35. <main>: Specifies the main content of a document.
  36. <map>: Defines a client-side image map.
  37. <mark>: Defines marked or highlighted text.
  38. <nav>: Defines a group of navigation links.
  39. <ol>: Defines an ordered (numbered) list.
  40. <option>: Defines an option in a select list.
  41. <output>: Represents the result of a calculation or user action.
  42. <p>: Defines a paragraph of text.
  43. <param>: Defines parameters for an <object> element.
  44. <picture>: Defines a container for multiple image resources.
  45. <pre>: Defines preformatted text, preserving both spaces and line breaks.
  46. <q>: Defines a short quotation.
  47. <rp>: Provides fallback parentheses for browsers that do not support ruby annotations.
  48. <rt>: Defines an explanation/pronunciation of characters (for East Asian typography).
  49. <s>: Defines text that is no longer correct, relevant, or needed (strikethrough).
  50. <script>: Defines client-side JavaScript within an HTML document.
  51. <section>: Defines a section in a document.
  52. <select>: Defines a drop-down list.
  53. <small>: Defines smaller text.
  54. <span>: Defines an inline container to style a portion of text.
  55. <strong>: Defines strong importance (typically displayed as bold).
  56. <style>: Contains style information for a document.
  57. <sub>: Defines subscripted text.
  58. <summary>: Defines a visible heading for a <details> element.
  59. <sup>: Defines superscripted text.
  60. <svg>: Defines a container for SVG graphics.
  61. <table>: Defines a table.
  62. <tbody>: Groups the body content in a table.
  63. <td>: Defines a cell in a table.
  64. <template>: Defines a container for holding HTML content not to be rendered when the page loads.
  65. <textarea>: Defines a multiline input control (a text area).
  66. <tfoot>: Groups the footer content in a table.
  67. <th>: Defines a header cell in a table.
  68. <thead>: Groups the header content in a table.
  69. <time>: Represents a specific period in time or a range of time.
  70. <title>: Defines the title of the document, usually displayed in the browser's title bar or tab.
  71. <tr>: Defines a row in a table.
  72. <track>: Provides text tracks (such as subtitles) for <video> or <audio> elements.
  73. <u>: Defines text that should be stylistically different from normal text (underlined).
  74. <ul>: Defines an unordered (bulleted) list.
  75. <var>: Defines a variable in programming or in a mathematical expression.
  76. <video>: Embeds video content in an HTML document.
  77. <wbr>: Specifies where in a text it would be ok to add line breaks.
